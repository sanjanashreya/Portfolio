Personal Portfolio Website

Project Overview
This is my personal portfolio website designed to showcase my skills, projects, and interests as a B.Tech student in Artificial Intelligence and Data Science.
The website features a modern UI with a video background, glowing effects, and a responsive layout. It highlights my work in AI, electronics, and intelligent systems.
